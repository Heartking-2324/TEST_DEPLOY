Flight Booking System

A simple yet powerful Flight Booking System built with Flask and SQLAlchemy.

Project Features:

  • Seat Selection (1-13)
  • Meal Preference (Veg / Non-Veg)
  • Extra Luggage Option
  • Payment Processing with Luhn Algorithm
  • Database Storage using SQLAlchemy
